# 本の世界転生オリキャラMaker Design

## Design Concept

古い魔導書を開き、ページをめくるたびに自分の転生後の姿が記録されていく体験にする。

方向性:

- 魔導書、羊皮紙、古いインク、金箔の罫線
- 暗すぎない、読める、静かな異世界ファンタジー
- 余白を残した日本語編集デザイン
- 過剰なゲームUIではなく、書物を読んでいる感覚
- 1ページずつ進むことで儀式感を作る

## Visual System

### Colors

- 背景: 深い藍黒、古い革表紙の影
- 紙面: 羊皮紙の生成り
- 文字: 焦げ茶、深いインク色
- 補助文字: 退色した茶灰
- 装飾線: 鈍い金、古い真鍮
- 選択中: 深い赤茶と金の組み合わせ

### Texture

CSSだけで以下を再現する。

- 紙面の薄いグラデーション
- 角の二重罫線
- ページ中央にわずかな影
- ボタンは金属ではなく、紙片や栞のように見せる

### UI Structure

- 左: 魔導書の質問ページ
- 右: 転生記録とPromptの記録ページ
- モバイル: 上から順に、導入、質問、転生記録
- 1問ずつ表示し、ページをめくる感覚を優先する

## Decorative Prompt

画像素材を作る場合の装飾プロンプト:

```txt
古い魔導書のUI装飾素材。
羊皮紙のような生成りの紙面、焦げ茶のインク、鈍い金の細い罫線、古い真鍮の角飾り。
中央に大きな余白を残し、日本語が読みやすい静かな異世界ファンタジーの雰囲気。
過剰な発光、派手な魔法陣、強いゲームUI感は避ける。
スマホ画面でも読みやすい、控えめで上品な装飾。
```

背景素材を作る場合:

```txt
古い革装丁の魔導書を思わせる背景。
深い藍黒と焦げ茶を基調に、紙の端、薄い影、鈍い金の細線だけで構成する。
本文やボタンを邪魔しない、暗すぎない、静かな余白のある背景。
ぼかしすぎず、装飾は画面端に控えめに配置する。
```

## Interaction

- 選択肢を押すと、転生記録に一行増える
- 次のページで章題が変わる
- おまかせは全ページを一気に埋め、最後のページへ移動する
- 閉じるは本を閉じる演出として最初に戻す

## Avoid

- 文字が読みにくい暗背景
- 派手な発光や魔法陣の多用
- 装飾過多でボタンが見えにくくなること
- 診断結果っぽい軽さだけで終わること
